# Backup, High Availability and Disaster Recovery

## Backup

- Built-in Azure SQL automatic backups with point-in-time restore (PITR).
- Long-term retention (LTR) configuration for clinical and regulatory requirements.
- Regular restore testing into non-production servers.

## High Availability

- Built-in regional HA capabilities of Azure SQL Database.
- No custom AG management required.

## Disaster Recovery

- Active geo-replication from UK South to UK West.
- Documented manual failover and failback runbooks.
- Scheduled DR testing and evidence capture.
